# **App Name**: CoLiveAssist

## Core Features:

- Preference-Based Roommate Matching: Use AI to analyze user preferences for gender, lifestyle habits, and other criteria to find the best roommate matches.
- Roommate Preference Input: Form to intake roommate preferences (gender, smoking, drinking, cleanliness, sleep, guests, cooking, noise).
- Room Preference Input: Form to intake room preferences (food, furnishing, bathroom, budget, location, amenities).
- Roommate Dashboard: Display rent split, chore assignments, and bill tracking with Indian names.
- Automated Payment Reminders: Send automated payment reminders via email/SMS to roommates.
- Group Chat: Enable a group chat feature for all roommates in a shared living space.
- Complaint Submission: Allow users to submit complaints that are then emailed to the house owner.

## Style Guidelines:

- Primary color: Deep purple (#673AB7), conveying trustworthiness.
- Background color: Very light purple (#F3E5F5).
- Accent color: Indigo (#3F51B5) to highlight interactive elements.
- Headline font: 'Poppins', a geometric sans-serif font for headings, capturing a precise and contemporary feel.
- Body font: 'Inter', a grotesque sans-serif font, providing a clean, modern aesthetic for the website content. 
- Use simple, outline-style icons related to household management and roommate interactions.
- Maintain a clean, card-based layout for displaying roommate profiles and management information.
